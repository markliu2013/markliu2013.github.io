**技术架构**  
- 后端：Java、Spring Boot、Spring Cloud Alibaba

- 网关与部署：Nginx、Docker、K8S、Jenkins
- 前端：Vue.js、Element UI
- 其他：RBAC 权限体系、雪花算法、DFA 敏感词过滤、Shiro

**主要职责**
- 基于 Spring Cloud Alibaba 进行微服务拆分，明确接口、策略、网关、日志、推送等模块边界
- 负责接口模块、策略模块、短信网关模块、搜索模块、推送模块、监控模块、后台管理模块的开发与联调
- 实现单条/批量短信发送接口及动态可插拔校验链（API Key、IP、签名、模板、余额、手机号格式、限流等），使用雪花算法生成全局唯一标识
- 完成号段补全、敏感词校验（DFA 算法 & Hutool）、黑名单校验、携号转网识别、验证码类短信限流、扣费与智能路由策略
- 实现状态报告推送客户方，支持延迟交换机与重试机制

- 集成 xxl-job 定时任务，监听队列消息与客户余额变化，异常告警。
- 基于 Elasticsearch 构建短信日志体系，支持海量日志的快速检索与问题追溯
